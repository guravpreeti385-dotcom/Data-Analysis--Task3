## Objective

The objective of this task is to analyze a Superstore dataset using SQL by performing data import, validation, filtering, aggregation, and exporting summarized results.

## Steps Performed

1. Created a new database and table with appropriate data types.
2. Imported the Superstore CSV dataset into MySQL.
3. Verified data using SELECT queries and record count checks.
4. Performed filtering using WHERE and sorting using ORDER BY.
5. Used aggregation functions like SUM, AVG, COUNT with GROUP BY.
6. Applied HAVING clause to filter grouped results.
7. Used LIKE for pattern matching and BETWEEN for date range analysis.
8. Exported final aggregated results into a CSV file using INTO OUTFILE.

The final output file `task3_output.csv` contains category-wise total profit summary.
